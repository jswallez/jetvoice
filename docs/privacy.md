# Privacy Policy

**Last updated:** January 2025

## Overview

Jetvoice is a privacy-focused voice transcription app. We collect no user data and store nothing on external servers.

## Data Collection

**We do not collect:**

- Personal information
- Usage analytics
- Crash reports
- Telemetry data

## Data Processing

### Voice Recordings

- Audio is recorded locally on your device
- Audio is sent directly to Google's Gemini API for transcription
- Audio is deleted immediately after transcription completes
- We do not have access to your recordings or transcriptions

### API Keys

- Your Gemini API key is stored in the macOS Keychain
- Keys are never transmitted to us
- Keys are only sent to Google's Gemini API for authentication

## Third-Party Services

### Google Gemini API

- Transcription is processed by Google
- Subject to [Google's Privacy Policy](https://policies.google.com/privacy)
- Audio data is sent securely via HTTPS

## Permissions

Jetvoice requires the following permissions, used solely for app functionality:

| Permission | Purpose |
|------------|---------|
| Microphone | Record your voice for transcription |
| Accessibility | Type transcribed text at your cursor |
| Input Monitoring | Detect global hotkey presses |

## Data Retention

- No audio data is retained after transcription
- No transcription text is stored
- Settings are stored locally in macOS preferences
- API keys are stored in your local Keychain

## Your Rights

Since we don't collect any data, there is no personal data to access, correct, or delete. Your API key and settings are stored entirely on your device and can be removed by uninstalling the app.

## Changes

We may update this privacy policy from time to time. Continued use of the app constitutes acceptance of any changes.

## Contact

Questions about privacy? Open an issue on [GitHub](https://github.com/jswallez/jetvoice).
